# Corcu

Corcu es una herramienta para borrar mensajes de Discord de forma masiva desde el navegador, basada en el proyecto antiguo Undiscord y modernizada para un funcionamiento más estable en 2026.

Este proyecto ha sido modificado, actualizado y rediseñado por Corcu.

---

## Características

- Eliminación masiva de mensajes propios
- Selección de servidor y canal
- Filtros por fecha
- Control de intervalos para evitar bloqueos
- Auto-scroll integrado
- Interfaz modernizada
- Funciona directamente desde Discord Web con Tampermonkey

---

## Requisitos

- Navegador compatible (Chrome, Edge o Firefox)
- Extensión Tampermonkey o Violentmonkey
- Acceso a Discord desde navegador (discord.com)

---

## Instalación

1. Instala Tampermonkey o Violentmonkey en tu navegador
2. Crea un nuevo script en el panel de la extensión
3. Pega el código del script
4. Guarda los cambios
5. Entra en Discord Web y recarga la página

---

## Uso

1. Abre Discord en el navegador
2. El panel de Corcu aparecerá automáticamente
3. Selecciona el canal o servidor donde quieres borrar mensajes
4. Configura filtros si es necesario (autor, fechas, intervalos)
5. Inicia el proceso
